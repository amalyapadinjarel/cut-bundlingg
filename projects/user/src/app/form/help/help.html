Logic to calculate effective org access

If org access is configured only via User application,

  • Effective org access is populated from the org access in User application.
  • Default division and facility is populated from the configuration provided in the User application if it is specified in the user application.

If org access is configured only via Roles application,

  • Effective org access is populated from all roles accessible to the user based on the org access provided in the Roles application.
  • Default division and facility is populated from the configuration provided in the roles.

If org access is configured in both User and Roles application,

  • Then the user will have access to the division and facilities which are common to both the User and Roles application.
  • Default division and facility is populated from the configuration provided in the User application if it is specified in the user application.
